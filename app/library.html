Views

The app has three main views, accessible via the tabs at the top:

  • Characters: Your local library. Browse, search, filter, and manage all your character cards
  • Chats: Chat history browser. View all chat logs across characters, search by content, and see which AI model was used
  • Online: Browse and import characters from online providers (ChubAI, JanitorAI, CharacterTavern, Pygmalion, Wyvern)

On mobile, swipe left/right on the main screen to switch between views.

Character Details

Click any character card to open their detail modal with tabs:

  • Details: Card fields (description, first message, personality, etc.), creator notes, and alternate greetings
  • Edit: Edit all card fields directly. Click Unlock Editing first to prevent accidental changes
  • Chats: All conversations with this character, with message counts and last-message preview. Resume any chat directly
  • Gallery: Images, GIFs, video, and audio. Download embedded media and provider gallery images
  • Related: Find similar characters in your library by shared tags, creator, and content overlap
  • Versions: Local snapshots and remote version history with diff preview (shown when history exists)
  • Info: Technical metadata: gallery ID, version UID, spec version, provider link details, extensions (enable in Settings)

Context Menu

Right-click any character card (or long-press on mobile) for quick actions:

  • Open Character, Favorite/Unfavorite, Select for Batch
  • Link to Provider / Provider Info, Check for Updates
  • Version History, View Gallery, Export, Delete

When multiple characters are selected, the context menu shows bulk actions instead.

Mobile Tips

  • Tap the avatar in the detail modal header for a full-size quick-view
  • Swipe left/right on character details to switch between tabs
  • Swipe left/right to cycle through alternate greetings
  • Long-press a character card to open the context menu (bottom sheet)
  • Swipe left/right on the main screen to switch views (Characters ↔ Chats ↔ Online)

Search Filters

Use special search prefixes to filter your character library:

creator:name Filter by author/creator name
linked:yes / linked:no Filter characters linked/not linked to any provider
chub:yes / chub:no Filter by ChubAI link status
janny:yes / janny:no Filter by JanitorAI link status
ct:yes / ct:no Filter by CharacterTavern link status
pygmalion:yes / pygmalion:no Filter by Pygmalion link status
wyvern:yes / wyvern:no Filter by Wyvern link status
favorite:yes / fav:no Filter by favorite status
version:text Filter by character_version field (version:none for empty)
gallery:id Filter by gallery_id (gallery:none for characters without one)
uid:id Filter by version_uid (uid:none for characters without one)

Combine prefixes with each other and with regular text: creator:john linked:yes dark elf finds linked characters by "john" matching "dark elf".

Tag Filtering

Click tags in the filter panel to cycle through states:

  • Green - Include: only show characters with this tag
  • Red - Exclude: hide characters with this tag
  • Gray - Neutral: no filter applied

Multiple tag filters combine: include tags use OR logic (match any), exclude tags are all applied.

Batch Tagging

Add or remove tags from multiple characters at once:

  • Desktop: Press Space or click the multi-select button in the toolbar to enter selection mode
  • Mobile: Long-press a card → Select for Batch, or open the More menu → Select Mode
  • Click characters to select/deselect, then click Edit Tags in the selection toolbar
  • The tag editor shows all tags across selected characters with counts; you can see which tags are shared by all vs. only some
  • Type new tags to add, or click existing tags to remove them

Provider Linking

Link your local characters to their online source for update checking, version history, and gallery downloads. Supported providers:

  • ChubAI: Card updates, gallery downloads, full Git version history
  • JanitorAI: Card linking and updates
  • CharacterTavern: Card linking and updates
  • Pygmalion: Card linking, updates, and gallery downloads
  • Wyvern: Card linking, updates, and gallery downloads

How to link: Click the Link indicator in the character detail sidebar, or right-click → Link to Provider. Search by name across all providers, or paste a direct URL.

Once linked:

  • Check for Updates: Compare local vs. remote card fields with a diff preview
  • Gallery Downloads: Download the character's provider gallery images
  • Version History: Browse published version history and restore any version
  • Quick Navigation: Jump to the provider page or preview in-app

Auto-Link

Bulk-link your entire library automatically:

  • Open from More Options (⋮) → Auto-Link Characters
  • Scans all unlinked characters by searching each provider's API by name
  • Results are categorized: Confident (strong match, auto-selected), Uncertain (multiple matches, you pick), and No Match
  • Review all matches before applying. Nothing is saved until you confirm
  • The scan is resumable: stop and resume later, or close and reopen without losing progress

Online Browsing

The Online tab lets you browse and import characters from providers:

  • Switch between providers using the dropdown selector
  • Search, sort, and filter by tags/categories. Each provider has its own options
  • Toggle NSFW content visibility per provider
  • Cards already in your library are marked with an "In Library" badge
  • Click a card to preview the full character details, then import with one click
  • Some providers support login for additional features (favorites, following lists)

Configure provider order and NSFW defaults in Settings → Online.

Version History

Track changes to your character cards with snapshots and diffs:

  • Auto-backup: A snapshot is saved automatically before every edit, restore, or card update, so you can always undo
  • Manual snapshots: Save the current state of any character at any time
  • Diff preview: Side-by-side comparison for every card field with added/removed highlighting
  • Remote versions: For linked characters, browse the full published version history and restore any version

How ChubAI remote versions work:

  • "Chub Page" entry: the current state on ChubAI, using the metadata API (with linked lorebooks resolved from the V4 Git API)
  • Git version history: each commit fetched from the V4 Git API (card.json), including the full merged lorebook

Configure max auto-backup count in Settings → General → Version History.

Card Editing

Edit all character card fields directly in the library:

  • Open a character → switch to the Edit tab → click Unlock Editing
  • All fields are locked by default to prevent accidental changes
  • Editable fields: name, description, personality, scenario, first message, alternate greetings, example messages, system prompt, post-history instructions, creator, creator notes, character version, tags, and the embedded lorebook
  • Use the expand buttons for full-screen editing of long text fields
  • A snapshot is auto-saved before each edit so you can always undo

Lorebooks & Card Updates

When a character card includes a lorebook (World Info), SillyTavern stores it in two places:

  • Embedded lorebook (inside the card): created at import time, not modified by SillyTavern's World Info editor
  • World Info file (in /worlds/): the live working copy used in chats. All your edits go here

These two copies are independent and diverge after import.

During a card update check:

  • The card's embedded lorebook is compared against the creator's latest version on the provider
  • New, modified, and local-only entries are shown in the diff
  • Your World Info file is never modified by the update checker

When you apply a lorebook update:

  • The card's embedded lorebook is replaced with the latest version
  • Local-only entries (in your card but not on the provider) are removed from the embedded copy
  • Your World Info file is not touched

Safety check for local-only entries:

  • Safe: also exists in your World Info file
  • Warning: not found in your World Info file, will be permanently lost

TL;DR: Updates only change the card's embedded data. Your World Info file (the copy that matters for chats) is never modified.

Duplicate Detection

Find potential duplicates in your library:

  • Open from More Options (⋮) → Find Duplicates
  • Compares character name, creator, description, tags, and lorebook entries
  • Shows a similarity score (0-100%) for each match
  • Review matches and delete unwanted duplicates

Adjust the minimum score threshold in Settings to control sensitivity.

Gallery Viewer

Full-screen media viewer for character gallery images, GIFs, and videos:

  • Open from right-click → View Gallery, or from the character detail modal
  • Navigate with arrow keys (← →), click left/right halves of the image, or use the prev/next buttons
  • Scroll wheel to zoom (up to 5×), drag to pan when zoomed, click or press 0 to reset zoom
  • Thumbnail strip at the bottom for quick jumping between images
  • Supports images, GIFs, and video (mp4/webm). Press Escape to close

Media Localization

Download remote media (images, audio, video) referenced in character cards for offline access:

  • Scans card text fields for remote URLs and downloads each file to the character's gallery folder
  • Also downloads gallery images from linked providers
  • Deduplication by file hash; won't re-download files you already have
  • Some hosts require SillyTavern's CORS Proxy to be enabled (User Settings → Network)

Enable "Replace remote media URLs" in Settings to auto-swap URLs when viewing. Use Bulk Localization in Settings → Media to process your entire library at once.

Gallery Sync

When Unique Gallery Folders is enabled (Settings → Gallery Folders), each character gets their own gallery folder even if they share names.

  • Gallery Sync audits the integrity of folder mappings on startup
  • Identifies: missing gallery IDs, broken folder mappings, orphaned mappings for deleted characters
  • View the audit status from More Options (⋮) → Gallery Sync Status
  • Can auto-repair missing IDs and mappings

AI Recommender

Get AI-powered character recommendations based on a natural-language prompt:

  • Open from More Options (⋮) → Card Recommender
  • Describe what you're looking for (e.g. "cozy fantasy characters", "dark horror scenarios")
  • Samples characters from your library and sends their metadata to an LLM
  • Works with your active SillyTavern Chat Completion source, or a custom OpenAI-compatible API
  • Batch Mode for wider library coverage: splits the pool across multiple LLM calls
  • Filter the sample pool by favorites, chat history, date range, and tags

The recommender only sends character names, tags, creator, and taglines, not conversation history or full card text.

Multi-Select & Batch Operations

Select multiple characters for bulk actions:

  • Desktop: Press Space or click the multi-select button, then click cards to select
  • Mobile: Long-press a card → Select for Batch, or More menu → Select Mode
  • Press Escape to exit selection mode

Available bulk actions:

  • Edit Tags: Batch add/remove tags across all selected characters
  • Check for Updates: Check all selected linked characters for remote updates
  • Favorite / Unfavorite: Toggle favorites in bulk
  • Export: Export multiple characters at once
  • Delete: Remove multiple characters

Importing Characters

  • From URL: Paste one or more character URLs (one per line) from any supported provider
  • From PNG: Drag & drop or browse local character card PNG files
  • From Online tab: Browse providers and import with one click from the preview modal
  • Progress tracking and error logging for multi-character imports
  • After import, characters are automatically checked for duplicates and can be auto-linked to their source

Chat History Browser

The Chats tab lets you browse all chat histories across all characters:

  • Two view modes: flat grid (all chats as cards) or grouped by character
  • Sort by date, character name, message count, chat length, or most active character
  • Each chat card shows character name, last message preview, message count, date, and AI model badge
  • Click to preview the full conversation with rendered messages
  • Open chats directly in SillyTavern or delete them

More Options Menu

The ⋮ More Options button in the topbar contains additional tools:

  • Import Characters: Import from URL or local PNG file
  • Find Duplicates: Scan your library for potential duplicate characters
  • Auto-Link Characters: Bulk-link unlinked characters to providers
  • Card Recommender: AI-powered character recommendations
  • Gallery Sync Status: Audit gallery folder mappings
  • Help & Tips: This modal
  • Gallery Settings: Configure gallery folders, media localization, and provider settings
